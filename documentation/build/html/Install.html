Installation¶

Fedoo can be installed on any plateform as soon as you can install a python 3 distribution with numpy and scipy and a subversion (SVN) manager.

Installation on Linux (Ubuntu, Debian or Mint)¶

  1. Install python 3 if not already installed:

$ sudo apt-get install python3

  1. Install numpy and scipy if not already installed

$ sudo apt-get update
$ sudo apt-install python3-pip
$ pip3 install numpy
$ pip3 install scipy

  1. Install the subversion manager

$ sudo apt-get install subversion

  1. Download the FEDOO library from https://ff-m2p.cnrs.fr/projects/fedoo/

$ cd where_you_whant_to_download_fedoo
$ svn checkout https://scm.ff-m2p.cnrs.fr/anonscm/svn/fedoo/

  1. Optional: if you want to be able to access fedoo from anywhere you can copy the fedoo directory to “<pythondir>/Lib/site-packages” or add the fedoo directory to the PYTHOPATH variable

Installation on Windows¶

  1. Install python, numpy and scipy. The easiest way is to install the ANADONDA distribution (https://www.anaconda.com/) which include by default numpy and and scipy with many other usefull libraries (recommanded).

  2. Download the FEDOO library from https://scm.ff-m2p.cnrs.fr/anonscm/svn/fedoo/ using any SVN utility (recomanded: https://tortoisesvn.net/)

  3. Optional: if you want to be able to access fedoo from anywhere you can copy the fedoo directory to “<pythondir>/Lib/site-packages” or add the fedoo directory to the PYTHOPATH environment variable.
